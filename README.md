# Pomodoro

A simple Pomodoro timer application built with the technologies already included in this project.

Technologies used:
- React
- TypeScript
- Vite
- ESLint
- @vitejs/plugin-react-swc

Getting started:

1. Install dependencies:

   ```
   npm install
   ```

2. Start development server:

   ```
   npm run dev
   ```

3. Build for production:

   ```
   npm run build
   ```

4. Preview the production build:

   ```
   npm run preview
   ```
